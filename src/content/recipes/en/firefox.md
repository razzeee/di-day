# Firefox Browser

Firefox is an open-source browser from Mozilla that protects your privacy.

## Preparation

1. **Download Firefox**

   - Visit [mozilla.org/firefox](https://www.mozilla.org/firefox/)
   - Download the version for your system
   - Install Firefox

2. **Import Data**

   - On the first start, Firefox offers to import bookmarks and passwords
   - Select your previous browser
   - Confirm the import

3. **Customize Firefox**

   - Sign in with a Firefox account (optional)
   - Adjust privacy settings
   - Install useful add-ons like uBlock Origin

4. **Set as Default Browser**
   - Firefox asks on the first start if it should be the default browser
   - Or: System Settings → Default Apps → Web Browser

## Dessert

**Sync**: With a Firefox account, you can synchronize bookmarks, passwords, and tabs between devices.

**Containers**: Firefox offers "Multi-Account Containers" to separate different identities (private, work, shopping).

## Topping

- **Tracking Protection**: Blocks many trackers by default
- **Open Source**: The source code is publicly viewable
- **Independent**: Firefox is developed by the non-profit Mozilla Foundation
- **Extensions**: Wide selection of privacy add-ons available
