# Firefox Browser
Firefox ist ein quelloffener Browser von Mozilla, der deine Privatsphäre schützt.
## Preparation
1. **Firefox herunterladen**
   - Besuche [mozilla.org/firefox](https://www.mozilla.org/de/firefox/)
   - Lade die passende Version für dein System herunter
   - Installiere Firefox
2. **Daten importieren**
   - Beim ersten Start bietet Firefox an, Lesezeichen und Passwörter zu importieren
   - Wähle deinen bisherigen Browser aus
   - Bestätige den Import
3. **Firefox anpassen**
   - Melde dich mit einem Firefox-Konto an (optional)
   - Passe die Datenschutzeinstellungen an
   - Installiere nützliche Add-ons wie uBlock Origin
4. **Als Standardbrowser festlegen**
   - Firefox fragt beim ersten Start, ob er Standardbrowser werden soll
   - Oder: Systemeinstellungen → Standard-Apps → Webbrowser

## Dessert
**Sync**: Mit einem Firefox-Konto kannst du Lesezeichen, Passwörter und Tabs zwischen Geräten synchronisieren.

**Container**: Firefox bietet "Container Tabs", um verschiedene Identitäten (privat, Arbeit, Shopping) zu trennen.
## Topping
- **Tracking-Schutz**: Blockiert standardmäßig viele Tracker
- **Open Source**: Der Quellcode ist öffentlich einsehbar
- **Unabhängig**: Firefox wird von der gemeinnützigen Mozilla Foundation entwickelt
- **Erweiterungen**: Große Auswahl an Datenschutz-Add-ons verfügbar
